# SDuX Vault — StackBlitz Examples

Live, runnable examples demonstrating [SDuX Vault](https://www.sdux-vault.com) state management.

These examples are formatted to run on [StackBlitz](https://stackblitz.com) and are linked from the SDuX Vault documentation site via the StackBlitz SDK API.

## Running Locally

```bash
cd stackblitz/<language>/<demo>
npm install
npm start
```

## Supported Language Demos

SDuX Vault is framework-agnostic and runs in any JavaScript or TypeScript environment. The table below tracks all supported languages, their demo priority, and current implementation status. All examples are written first in Angular and then converted to each target language.

These languages have working StackBlitz examples available in the `stackblitz/` directory and are linked from the SDuX Vault documentation site.
